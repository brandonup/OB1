# Brain — Failure Log

**Purpose:** Track where Brain fails during the 2-week validation period. This log drives BRA-9 (Phase 3 gate) — what extensions to build and what structure Brain actually needs.

**Validation window:** 2026-04-06 → 2026-04-20

---

## What counts as a failure

| Type | Description |
|---|---|
| `retrieval-miss` | Searched for something I know I captured — didn't find it |
| `capture-friction` | Wanted to capture something but the moment passed |
| `stale` | Retrieved a thought but it was outdated or wrong |
| `missed-surface` | Relevant thought existed but wasn't surfaced when it should have been |
| `false-positive` | Search returned irrelevant results, buried the real match |
| `context-repeat` | Had to re-explain context to the AI that should already be in Brain |

**Severity:** 1 = minor annoyance · 2 = slowed me down · 3 = blocked or cost me something real
